# Instructions

Must have a working **nix** and **direnv** install.

1. Run `direnv allow` at the project's root dir.
2. Start whatever server you want to with `python -m http.server 8000` after cding
to the desired server's dir.
3. Run your desired `*_main.py` file. 

If you don't have Nix then that's not my problem.
